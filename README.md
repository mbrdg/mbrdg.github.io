# Coffee & State Synchronization

This repo holds all the contents related to my personal website.

The site is hosted through [GitHub pages](https://pges.github.com)
and it is built with [Astro](https://astro.build).

## Dependencies

I use `bun` instead of node as the javascript runtime environment.
Therefore, I recommend to see the docs on how to use `bun` together with Astro.

## Commands

All commands are run from the root of the project, from a terminal:

| Command                      | Action                                           |
| :--------------------------- | :----------------------------------------------- |
| `bun install`                | Installs dependencies                            |
| `bunx --bun astro dev`       | Starts local dev server at `localhost:4321`      |
| `bunx --bun astro build`     | Build your production site to `./dist/`          |
| `bunx --bun astro preview`   | Preview your build locally, before deploying     |
| `bunx --bun astro ...`       | Run CLI commands like `astro add`, `astro check` |
| `bunx --bun astro -- --help` | Get help using the Astro CLI                     |

And that should be everything!
For more information see the Astro's [documentation](https://docs.astro.build).
